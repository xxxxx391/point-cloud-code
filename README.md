# 点云脚本(包含了点云处理的各种脚本)

## 1.点云脚本描述
- transform.py:输入：点云数据（ply格式）和变换矩阵（txt格式），输出：点云数据（ply格式）；
- voxel_subsample.py：对输入点云进行下采样和尺度变换（先进行下采样再进行尺度变换）；
- voxel_subsample_1.py：对输入点云进行下采样和尺度变换（先进行尺度变换再进行下采样）；
- pre_analyse.py：对点云的变换矩阵进行精度分析；
- point_plane_seg.py：对点云进行分割，分割出平面点云；输入：ply原始点云数据，输出：分割出平面的点云数据；
- image_circle_points.py：提取靶标红外图像中的圆心坐标，并可视化和打印；输入：输入为红外图像路径，输出：包含检测结果的图像和打印的坐标；
- pnp_solve：根据图像特征点坐标和点云特征点坐标计算变换矩阵；输入：包含坐标的txt文件，输出：变换矩阵、将结果投影至图片上的特征点；
